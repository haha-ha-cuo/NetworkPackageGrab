# NetworkPackageGrab

这是一个使用 C++ 编写的网络抓包工具。

## 功能

- 抓取网络数据包和流量监控
- 解析和显示数据包信息

## 依赖

- [npcap](https://npcap.com/) - Windows 平台下的抓包库
- [cpr](https://github.com/libcpr/cpr) - C++ HTTP 请求库
- [OpenSSL](https://www.openssl.org/) - TLS/SSL 和加密库
- [C++ REST SDK (cpprestsdk)](https://github.com/microsoft/cpprestsdk) - 异步 HTTP 客户端和服务器库
